Project Description
This project uses Machine Learning to predict house prices based on various features like overall quality, living area, basement size, garage capacity, year built, and neighborhood.

Features Used
1. Overall Quality
2. Above Ground Living Area
3. Total Basement Area
4. Garage Cars
5. Year Built
6. Neighborhood

Algorithm Used
Linear Regression

Technologies Used
- Python
- Flask
- scikit-learn
- Pandas
- NumPy
- Joblib

How to Run Locally
1. Clone this repository
2. Install dependencies: `pip install -r requirements.txt`
3. Run the app: `python app.py`
4. Open browser to `http://127.0.0.1:5000`
